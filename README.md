# CineWeb  
Sistema web para gerenciamento de cinema desenvolvido com **React + TypeScript + Vite**.

O projeto faz parte de um laboratório prático e inclui funcionalidades de cadastro, listagem e controle de exibições em um cinema, utilizando **JSON-Server** como backend fake.

---

## Tecnologias Utilizadas
- React + TypeScript  
- Vite  
- Bootstrap 5  
- React Router DOM  
- Zod (validação de formulários)  
- JSON-Server (API simulada)

---

## Funcionalidades
### **Filmes**
- Cadastro de filmes  
- Validações (título, sinopse, duração etc.)  
- Listagem e exclusão  

### **Salas**
- Cadastro de salas  
- Capacidade, número da sala  
- Listagem  

### **Sessões**
- Agendamento de sessões  
- Seleção de filme e sala  
- Validação de data futura  
- Listagem de sessões existentes  

### **Ingressos**
- Venda de ingressos (inteira / meia)  
- Registros associados à sessão selecionada  

### **Comandos de Execussão**
- npm run dev
- npm run dev
